## YOGA DAILY

## Description
Yoga Daily is a mobile application built for physical fitness. Here, one can discover many Yoga training sessions and select accordingly. It will provide the user ideal time and animation for selected yoga. User can pause, restart  and can quit the yoga session.


##  Technology Used
The application is built using the following technologies :\
Flutter SDK\
Dart Programming language\
SQFlite\
Firebase Database\
Provider for state management
